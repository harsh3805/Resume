# Harsh Vardhan Resume

This repository contains a GitHub Pages-ready resume website.

## Files

- `harsh.html` - main resume website
- `resume.pdf` - downloadable PDF resume

## How to publish on GitHub Pages

1. Create a public GitHub repository named `resume` or `portfolio-resume`.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select branch **main** and folder **/root**, then click **Save**.
6. Your resume will be visible at `https://YOUR_USERNAME.github.io/REPOSITORY_NAME/`.
